## 1. 서비스 소개
### 개요
네이버 예약 정보 모아보기 서비스, 낙낙
백엔드 레포지토리

### 주요 기능
- React 활용 페이지 및 컴포넌트 구현
- Selenum을 활용한 타겟 사이트 정보 크롤링
- JPA 활용 데이터베이스 구축
